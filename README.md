## 참고자료
교재 Spring Security in Action(로텐티우 스필카)를 보고 예제코드를 작성하거나,  이를 활용하여 보안 로직을 만들어 보았습니다.
이 프로젝트는 Spring Boot 3를 기반으로 Spring Security의 핵심 개념과 기능을 단계별로 학습하며 구현한 내용입니다.

---
